# CNVD_Modify
适用于CNVD的篡改猴（Tampermonkey、Greasemonkey）脚本,按需自取，有需求、bug提issues，有时间就处理。

## 作用
- 漏洞列表页添加复制单个或所有漏洞标题
- 优化用户中心已提交漏洞展示和复制漏洞编号
